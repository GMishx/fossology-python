Fossology Report¶

Methods used to access “report/” endpoints.

class fossology.report.Report¶

Class dedicated to all “report” related endpoints

download_report(report_id, as_zip=False)¶

Download a report

API Endpoint: GET /report/{id}

FIXME: Accept header “application/zip” does not work in version 1.0.6

Parameters

  • report_id (int) – the id of the generated report

  • as_zip (boolean) – control if the report should be generated as ZIP file (default False)

Raises

FossologyApiError – if the REST call failed

generate_report(upload, Format=None)¶

Generate a report for a given upload

API Endpoint: GET /report

Parameters

  • upload (Upload) – the upload which report will be generated

  • Format (ReportFormat) – the report format

Returns

the report download url

Return type

list of Job

Raises

FossologyApiError – if the REST call failed
